# Cybersecurity Home Lab

## Project Overview

This project demonstrates the setup and configuration of a cybersecurity home lab using VirtualBox. The lab consists of Kali Linux and Ubuntu virtual machines connected through a controlled virtual environment for security testing and learning purposes.

## Objectives

* Build a virtualized cybersecurity environment
* Perform network reconnaissance using Nmap
* Configure secure remote access using SSH
* Analyze network traffic using Wireshark
* Practice Linux administration and networking concepts

## Lab Environment

### Operating Systems

* Kali Linux
* Ubuntu Linux

### Tools Used

* Oracle VirtualBox
* Nmap
* Wireshark
* SSH
* Linux Terminal

## Activities Performed

### Network Scanning

Performed host discovery and network reconnaissance using Nmap.

### Remote Access

Configured and tested SSH connectivity between virtual machines.

### Packet Analysis

Captured and analyzed network traffic using Wireshark.

### System Administration

Performed Linux system configuration and troubleshooting tasks.

## Skills Demonstrated

* Network Security Fundamentals
* Network Scanning
* Packet Analysis
* Linux Administration
* Virtualization
* SSH Configuration
* Troubleshooting
